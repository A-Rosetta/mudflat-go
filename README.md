# Mudflat Go! 滩涂大冒险

深圳湿地生态文旅打卡平台前端交互原型。项目使用原生 HTML、CSS 和 JavaScript，可直接部署到 Cloudflare Pages。

## 本地预览

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/`。

## Cloudflare Pages

- Framework preset: `None`
- Build command: 留空
- Build output directory: `/`
- Production branch: `main`
